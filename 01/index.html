Структура найпростішого html-документу

<!DOCTYPE HTML PUBLIC> - специфікація версії HTML
<html>- контейнер содержания Веб-страницы
<head> - голова документа, предназначенная для браузера и поисковых систем
<title> Сайт про HTML </title> - название документа, что отображается в верхней части окна браузера
</head> - конец головы документа
<body> - тело документа, сохраняет содержимое веб-страницы
</body>- конец тела документа
</html> - конец содержания Веб-страницы
